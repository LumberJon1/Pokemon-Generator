Pokemon Generator

Instructions

Welcome to Pokemon Generator!

Click the "Randomize" button below to generate a random pokemon. Its number will appear in the box labeled "Pokemon."

The button will also generate two random types for the pokemon, which can be used as a prompt to reimagine and draw that pokemon as one or both of those new types.

If the original pokemon has only one type, feel free to use only one of the randomly-generated types when you draw the new version of the pokemon.
